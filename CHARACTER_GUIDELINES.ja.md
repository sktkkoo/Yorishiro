# Yori キャラクターガイドライン

*[English](CHARACTER_GUIDELINES.md)*

Yori は Yorishiro に同梱されるキャラクターです。このガイドラインは、Yori というキャラクターをどう使ってよいかを示すものです。ファン活動を制限するためではなく、歓迎するために置いています。

## 自由にどうぞ（許諾不要）

- **ファンアート・ファン作品の制作** — イラスト、漫画、小説などの二次創作。
- **スクリーンショット・クリップ・動画・配信の投稿** — Yori が映る Yorishiro の様子を投稿すること。広告収益のある YouTube / Twitch チャンネルやメンバーシップなど、**収益化されたプラットフォームでの投稿も含みます**。
- **商用の文脈で Yorishiro と Yori を使うこと** — 職場での利用、配信での利用など。
- **記事・登壇資料・教材でのスクリーンショットやクリップの利用。**

- Yori の**性的・露骨な表現**（VRM モデル埋め込みのライセンスでも禁止されています）。
- Yori が製品・サービス・意見を**公式に推奨しているかのように見せる**コンテンツ。
