# CLAI キャラクターガイドライン

*[English](CHARACTER_GUIDELINES.md)*

CLAI は Yorishiro に同梱されるキャラクターです。このガイドラインは、CLAI というキャラクターをどう使ってよいかを示すものです。ファン活動を制限するためではなく、歓迎するために置いています。

## 権利について

キャラクターデザイン・VRM モデル制作：LUCAS 様（[@lucas_VTuber](https://x.com/lucas_VTuber)）。本モデルの著作権は Yorishiro プロジェクトに譲渡されており、著作者人格権は LUCAS 様に帰属します。詳細は [`CREDITS.ja.md`](CREDITS.ja.md) を参照してください。

CLAI というキャラクターおよびその VRM モデルは、リポジトリの MIT ライセンスの**対象外**です。MIT ライセンスはソースコードにのみ適用されます。

## 自由にどうぞ（許諾不要）

- **ファンアート・ファン作品の制作** — イラスト、漫画、小説などの二次創作。
- **スクリーンショット・クリップ・動画・配信の投稿** — CLAI が映る Yorishiro の様子を投稿すること。広告収益のある YouTube / Twitch チャンネルやメンバーシップなど、**収益化されたプラットフォームでの投稿も含みます**。
- **商用の文脈で Yorishiro と CLAI を使うこと** — 職場での利用、配信での利用など。
- **記事・登壇資料・教材でのスクリーンショットやクリップの利用。**

クレジットは任意ですが歓迎します：`CLAI © Yorishiro project`

## 事前にご相談ください

以下は [GitHub Discussions](https://github.com/sktkkoo/Yorishiro/discussions) で事前にご連絡ください：

- CLAI を用いた**グッズ・商品の販売**（物理・デジタルとも）。
- CLAI を**別の製品・サービス・ブランドの顔やマスコットとして使うこと**。
- **VRM モデル自体を Yorishiro の外で抽出・再配布・転用すること** — モデルの再アップロード、他プラットフォームでのアバター利用、他ランタイム向けの変換など。（これはモデルファイルに埋め込まれたライセンスでも禁止されています。）

## 禁止事項

- CLAI の**性的・露骨な表現**（VRM モデル埋め込みのライセンスでも禁止されています）。
- CLAI が製品・サービス・意見を**公式に推奨しているかのように見せる**コンテンツ。
